# How-much-are-you-Mellstroy-
Проект выполняет сравнение лиц пользователя с лицом Mellstroy на основе алгоритма распознавания лиц OpenCV — LBPH (Local Binary Patterns Histograms).
Он обучается на фото Mellstroy и оценивает схожесть любого нового лица, выдавая процент совпадения и шуточное текстовое заключение.
