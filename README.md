# How-much-are-you-Mellstroy-
Проект выполняет сравнение лиц пользователя с лицом Mellstroy на основе алгоритма распознавания лиц OpenCV — LBPH (Local Binary Patterns Histograms).
Он обучается на фото Mellstroy и оценивает схожесть любого нового лица, выдавая процент совпадения и шуточное текстовое заключение.

Как работает
	1.	Обнаружение лиц
Используется каскад Хаара (haarcascade_frontalface_default.xml) для поиска лиц на фото.
	2.	Обучение модели
Класс OpenCVMellstroy загружает все изображения из mellstroy_photo/, находит на них лица и обучает алгоритм cv2.face.LBPHFaceRecognizer.
	3.	Сравнение
Для каждого нового фото пользователя:
	•	Лицо выделяется из изображения.
	•	Модель вычисляет метрику схожести (confidence).
	•	Схожесть переводится в проценты:

similarity = 100 - confidence
